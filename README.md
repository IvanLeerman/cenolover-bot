# Ценоловер - Telegram бот для аукционов

Telegram-бот для проведения автоматических аукционов с интеграцией Google Sheets.

## Основные функции
- Регистрация пользователей
- Создание и управление лотами
- Система ставок с автообновлением
- Админ-панель
- Интеграция с Google Sheets
- Rate limiting
- Асинхронная работа

## Установка
1. Клонировать репозиторий
2. Установить зависимости: pip install -r requirements.txt
3. Скопировать config.py.example в config.py и заполнить
4. Запустить: python bot.py

## Конфигурация
- BOT_TOKEN - токен бота от @BotFather
- Данные PostgreSQL
- ID администраторов
- ID канала для публикаций

## Структура проекта
- bot.py - основной файл бота
- async_db.py - работа с базой данных
- payment.py - платежная система
- google_sheets.py - интеграция с Google Sheets
